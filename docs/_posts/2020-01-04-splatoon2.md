# パブロにハマってる

スプラトゥーン 2、最近はパブロというブキにドハマりしてます。勝率は全然良くないんですけど、楽しくて使うのをやめられない……使ってて一番楽しいブキは自分の中ではパブロに決まりです。連打はなぜか当初から最速で出せてる気がするので、そこの才能はあるっぽい。

実力のほうは、たぶん XP2,100 ~ 2,200 くらいしかないので、しばらく使い続けて XP2,300 くらいまでは安定して出せるくらいには持っていきたいです。好きなブキを一番得意なブキにしたいです。（今はデュアルがイチバンパワーが高い）
