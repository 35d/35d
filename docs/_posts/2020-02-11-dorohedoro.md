# ドロヘドロを見始めた

![ドロヘドロ](../assets/img/2020-02-11-1.jpg)

Netflix でテラスハウスを見ようとしたら残念ながら今週は更新されておらず、
たまたま近くにあった「ドロヘドロ」を見始めたのですが、メチャメチャ面白いですねこれ。

魔法によって頭を爬虫類に変えられた記憶喪失の男が、自分の本当の顔と記憶を取り戻す物語で、
ところどころ大人向けなギャグが混じっていたり、大迫力の戦闘シーンがあったりと、
THE 大人向けアニメっていう印象を受けました。

漫画の存在自体は大学生の頃から知っていて、面白いという噂は聞いたことはあったものの、今まで読んだことはなかったので、
この機会に見れて良かったです。今日は 4 話くらいまで見たので続きが楽しみです。
