# 「家に帰ると妻が必ず死んだふりをしています。」を見た

![「家に帰ると妻が必ず死んだふりをしています。」](../assets/img/2020-01-16-1.jpg)

ちょっと前ですが、Netflix でたまたま目に入ったので見てみました。

タイトルの通りですが、家に帰ると必ず死んだふりをしている妻（榮倉奈々）と、夫（安田顕）のハートフルコメディ。最後のオチのところがサイレントだったのは、意味深な感じに見せただけのただの怠慢なのでは？と思ってしまいました。演出もところどころ安っぽく感じてしまうところもあり、脚本・演出ともに映画の内容としては微妙でしたが、榮倉奈々がとにかく可愛かったので個人的には満点です。

大切な人を大切にしようと思える映画でした。

いやー榮倉奈々が可愛かった（２回目）
