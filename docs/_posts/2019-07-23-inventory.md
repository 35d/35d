# 棚卸し(2019 年 7 月)

気づいたら 2019 年も半分が終わってしまっているんですね。今月も棚卸しやっていきたいと思います。(タイトルが棚卸しってなっているけど、実質やってることは３ヶ月の振り返りです)

### 👩‍💻 お仕事 (React Native 等)

相変わらず仕事では React Native を書く機会が多いです。

[前回の棚卸し](https://medium.com/35d/%E6%A3%9A%E5%8D%B8%E3%81%97-2019%E5%B9%B44%E6%9C%88-4aa6086c26a7)のときに書いた Android 対応プロジェクトが一段落して、次のプロジェクトに取り掛かっている最中です。詳しいことは書けないですが、新しいプロジェクトの性質上、ネイティブのコードをいじる機会も増えてきたりしました。また、プロジェクトの合間に React Native のバージョンアップなど細かいことをやったりしました。

勉強会とかにもちょこちょこ顔を出すようになりました。今までずっとインターネットに引きこもっていましたが、外部の人と話すのは刺激があって面白いなと感じています。自分の興味関心が主に React Native とフロントエンド周りなので、そのへんの勉強会に顔を出してみたりしました。なんか良さげな勉強会あったら教えてください。一緒に行きましょう 🙌

- [【半年に一度の大規模会】ReactNative にゆかりのあるスタートアップが集う会 (2019/06/18 19:30〜)](https://r-n.connpass.com/event/132444/)
- [俺得フロントエンド (1) LT 会 (2019/06/12 19:00〜)](https://opt.connpass.com/event/130433/)

👆 こういうのに行ったりしてました。

### 🦑 YouTube (Splatoon)

最近のマインドシェアの一番はこれです。オンラインでゲームの配信(主にスプラトゥーン)をしたりしていて、チャンネル登録者数がだんだんと増えていく感覚がとても楽しくて、毎日ちまちまと続けています。チャンネル登録者数は 111 人まで増えました(2019/07/22 現在)。

- [にっけちゃんねる](https://www.youtube.com/channel/UC9DyjyQZBL74H7nCT3eG0Rg?view_as=subscriber&sub_confirmation=1)

おかげさまでスプラトゥーンはウデマエがとうとうオール X になって、プレイヤー人口の上位に入れるようになりました。また、今月は、会社の人とのリグマで上位に入ることが出来ました。

ゲームに限らずですが、**自分の頑張りが数値になって返ってくることに楽しさを感じます。**ウデマエが伸びたり、チャンネル登録者数が増えたりすると楽しいです。

この活動を通して、動画配信や映像制作に関してだいぶノウハウが溜まってきました。ゲームが軌道に乗って一段落したら、プログラミングの配信とかもできたらいいなーと目論んでいたりします。

### 🏡 引っ越し

5 月に引っ越しをしました。1 年ほど渋谷に住んで徒歩通勤してきましたが、

- 家が狭かった

- 街が汚すぎた

という２つの理由で引っ越しを決意しました。引越し先は新代田という街です。渋谷と比べてとても静かな街で落ち着いた毎日を過ごすことができています。これで東京に来てから３回目の引っ越し（明大前 → 下北沢 → 神泉 → 新代田）で、だんだんと引っ越しのプロになりつつあります。

この 3 ヶ月は今までの 3 ヶ月に比べてアウトプット量が少なかったようなような気がしています。次の 3 ヶ月はこれをやったぞと胸を張って言えるような活躍を残したい気持ちです。

### 💪 次の３ヶ月について

- 今没頭しているのは YouTube なので、そこで突き抜けたい気持ち

- プライベートで技術力磨けてないのでなんかやらなきゃという気持ち

- （今まで一人でやることが多かったので）一人でじゃなくて色んな人と一緒に色々やりたい気持ち

また３ヶ月後に書きますね。
