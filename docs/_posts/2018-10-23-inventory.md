# 棚卸し(2018 年 10 月)

[時雨堂さんの記事](https://medium.com/@voluntas/%E6%A3%9A%E5%8D%B8%E3%81%97-201810-46cb9b60743f) を読んだら自分もやりたくなったのでやってみる。遅くなっちゃったけど [今年の上半期に立てた目標](https://medium.com/35d/2018%E5%B9%B4%E4%B8%8A%E6%9C%9F-%E3%82%84%E3%82%8B%E3%81%93%E3%81%A8-%E3%82%84%E3%82%89%E3%81%AA%E3%81%84%E3%81%93%E3%81%A8-899580eae624) とも照らし合わせつつ振り返り。

## React Native

相変わらず仕事もプライベートも React Native ばっかり書いてる。おかげさまで Redux / Atomic Design / Flow / アニメーション あたりにだいぶ詳しくなった。世の中にアニメーション周りのドキュメントが少なすぎて業務でツライ思いをすることがたくさんあるので、同じ思いをする人が一人でも減れば良いなと思ってアニメーションの記事を書き始めた。今年中にあと 1 記事くらい出したい。

- [React Native の LayoutAnimation を使ったアニメーションが簡単で便利](https://medium.com/35d/react-native-%E3%81%AE-layoutanimation-%E3%82%92%E4%BD%BF%E3%81%A3%E3%81%9F%E3%82%A2%E3%83%8B%E3%83%A1%E3%83%BC%E3%82%B7%E3%83%A7%E3%83%B3%E3%81%8C%E7%B0%A1%E5%8D%98%E3%81%A7%E4%BE%BF%E5%88%A9-aac0cdc09dcf)

## Ruby on Rails

プライベート開発で作っている React Native アプリの API を Ruby on Rails で作っている。いままでなんとなく理解してたつもりだったけど、全然理解できてなかったことを知った。知らないということを知れる、という意味で良い経験になっているなあって感じがする。自分の興味が、堅牢な設計の実現とかそのへんにありそう、ってことを知れたのも収穫。
[Web API: The Good Parts](https://www.amazon.co.jp/Web-API-Parts-%E6%B0%B4%E9%87%8E-%E8%B2%B4%E6%98%8E/dp/4873116864/ref=sr_1_1?ie=UTF8&qid=1540190844&sr=8-1&keywords=web+api)

## Vue.js

ここ数ヶ月は Web の開発をほぼやってないので、特にアップデートなし。11 月に新しいサービスを作るのでそこで使う予定。

## Design

仕事で自分が手を動かすことは一切なくなった。プライベートで開発してるもののデザインをたまにやるくらい。今後はデザインする割合をどんどん減らしていっても良さそうだなって思ってる。何か作りたいなって思ったときはデザイナーの力を借りるようにしたい。たまに本は読む。

## Splatoon2

技術ぜんぜん関係ないけどだいぶ時間取ってるのでここにも書いておく。GW 中に買って、今日までで 500 時間くらいプレイしてる。「今月中にどこまで行く」みたいな目標を定めてストイックにやり続けてる。ただプレイしつづけるよりは何かアウトプットあった方が面白いのかなと思って、機材を揃えて 10 月頭に [YouTuber デビュー](https://www.youtube.com/channel/UC9DyjyQZBL74H7nCT3eG0Rg)もした。けっこうモチベーション高いのでしばらくやり続けると思う。12 月にスマブラも出るので、それもプレイ動画撮りたい気持ち。ちなみにいまガチエリアがウデマエ X で、それ以外がウデマエ S+0 っていう状態。

今年の頭に書いた記事を読み返してみると、半年間の目標の意味のなさに気付く。もうちょっとマメに振り返って軌道修正していかないとダメだなって思った。自分の興味の対象を知れたって意味では良い半年間だったように思う。

残り少ないけど、今年の残りも頑張りたい。
