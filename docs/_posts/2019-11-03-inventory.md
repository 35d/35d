# 棚卸し(2019 年 10 月)

定期的にやるアレです。

### お仕事・技術

メインのお仕事。スクラム開発でコツコツとプロダクト改善を続けてます。結構がっつりとスクラム導入して半年以上が経ちました。ここまでオレオレカスタマイズせずにスクラム続けられてるところは珍しいんじゃないかなと思ってます。おかげさまでだいぶスクラム筋が鍛えられつつある気がしています。

メインのお仕事と並行して、受託開発もちょこちょこやりました。NuxtJS で初めての案件をこなしました。Vue.js でガッツリとプロダクションコードを書いたのは初めてだったので良い経験になりました。パフォーマンスチューニングやと Vuex のストア設計が経験できたのが良かったです。やっぱりゴリゴリコードを書いてる時間が一番楽しい気がする。

一方で、コミュニケーションコストを減らすための工夫とか、バックエンドとの分業もうちょっといい感じにできたかなとか、いろいろと反省もある案件だったので、次回以降いい感じに工夫したいです。。。締め切り直前プチ炎上しかけてつらかったのも良い経験になりました w

ちょっとできる技術スタックがこんな感じになりました。

- React Native / React / Vue.js / NuxtJS / Ruby on Rails
- HTML / CSS / JavaScript
- Netlify / Firebase Hosting / Heroku
- CircleCI
