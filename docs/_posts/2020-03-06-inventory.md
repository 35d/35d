# 棚卸し(2020 年 3 月)

定期的にやるアレです。

1 月より本格的にスタートアップの立ち上げをやっています。

## NuxtJS + TypeScript

最近はひたすら NuxtJS と TypeScript でプロダクトを作っています。
静的なページも作れるし、動的なプロダクトも作れる NuxtJS とても便利で使い勝手が良くて、手に馴染んできました。

TypeScript はまだ良く分かっていないところが多いので、インプットしていかねばの気持ちです。早くマスターしたい。
