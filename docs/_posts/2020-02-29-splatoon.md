# 自分がプレイ中考えていることを言語化してみた（スプラトゥーン 2）

プレイしながらの解説と、後付けの字幕で解説の補足をしており、この両軸で言語化を試みてみました。

こうやって改めて言語化してみると、自分のダメなところとかも見えて面白いですね。
