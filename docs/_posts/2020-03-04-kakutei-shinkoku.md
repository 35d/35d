# 2019 年度の確定申告の提出をした

家から徒歩圏内に偶然にも税務署があったので、そこへ提出しに行きました。
提出自体は一瞬で、軽いチェックとハンコ推してもらったのみで、
「あれ？こんなんでいいんだっけ？」感がすごかったんですが、無事提出を終えてほっとしています。

無事終わってほっと一息なのと、ちゃんと経費とかは都度ちまちま入力していこうと決意した一日でした。
