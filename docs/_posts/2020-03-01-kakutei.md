# 2019年度の確定申告の書類準備をした

ここ数日で確定申告の準備をしました。あとは書類を提出しに行くだけ、という状態まで進めました。
徒歩圏内に税務署があるので、来週あたり直接提しに行こうかなと考えています。
青色が控除額が高いということを聞き、青色にチャレンジしてみました。

## どうやったの？
会計 freee を使って収入と経費を自分でポチポチ入力していきました。
行けるところまでは自分で進めてみて、
完成した書類は高校の後輩にレビューしてもらって細かい部分の修正をしたりしました。
クレカの明細とかをちまちま見ていって、経費になりそうなものについてはボタンを押していって、
現金で支払ったものについては手入力で行きました。


## どのくらいかかった？
未整理の状態から始めて、インプットやら領収書の整理とかも含めてざっと10時間くらいかかった気がします。
今年は仕事用の口座とプライベートの口座が分かれていなかったりしたのと、
そもそものインプットに時間がかかったような気がします。
（まだ完全には理解できていない部分もたくさんあります）

## 来年に向けて
- Web 系の支払い（サーバ代とかドメイン代とか諸々サブスクサービスとか）
の領収書の準備にけっこう時間がかかったので、どこで出力できるのか（メールで領収書が送られてきているのか、それともWebページにログインして領収書を吐き出すのか、等）をブログ記事にまとめておきたいと思います。
- できれば年度の途中からでも良いから、仕事の経費の出費は特定のクレジットカードからにした方がパッと済みそうだなと感じました。でも今契約してるサブスクサービスの支払先をまとめて変更する手間もなーとか考えると悩ましいところ。
- もしかしたら freee も契約しなくても良いかもしれないなとか考えましたが、請求書とか出力するのに便利だから結局使い続けることになりそうです。
